### 粘性组件

#### 场景

- 一般用作侧边栏
- 建议场景如按钮，回到主页等
- 简要使用，不包含复杂逻辑的独立场景



#### 设计说明

| 参数    | 说明         | 类型   | 可选值 | 默认 | 备注 |
| ------- | ------------ | ------ | ------ | ---- | ---- |
| top     | 距离顶部距离 | number |        | 0    |      |
| z-index | z-index      | number |        | 1    |      |



#### slot

| name     | 说明                           |
| -------- | ------------------------------ |
| 匿名插槽 | 其他展示的方式，默认为组件名称 |
